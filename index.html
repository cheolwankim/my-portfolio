About me

컴퓨터공학 전공을 바탕으로 웹 프론트엔드 개발에 집중하고 있습니다. 혼자 개발하더라도 GitHub Flow, 이슈 기반 개발, PR 리뷰 등 팀 개발 환경을 스스로 재현하면서 프로젝트를 진행해 왔습니다.

블로그, 게임 API 기반 검색 서비스, 지도 기반 검색 서비스 등 실제 사용 가능한 형태의 사이드 프로젝트를 직접 설계·구현·배포했습니다. 프론트엔드에 집중하되, Express와 MongoDB를 이용해 백엔드와 인증 흐름까지 이해하려고 했습니다.

단순히 화면을 만드는 것보다 어떤 구조가 유지보수와 확장에 유리한지, 어떻게 하면 협업에 방해되지 않는 코드를 쓸 수 있는지를 항상 고민하고 있습니다.
